## Project Description

An e-commerce platform designed to strengthen the business's competitive advantage by preserving the current website's core UI and essential e-commerce functionality while significantly enhancing the overall user experience and increasing customer trust.

The platform contains all fundamental e-commerce capabilities, including product management, shopping cart, order tracking, discounts, and a comprehensive administrative dashboard. 

We introduce modern features that improve client UX and trust:
### Key Enhancements

- **Semantic Search:** Allows customers to find relevant products using natural language instead of exact keywords, making product discovery faster and more intuitive.

- **AI Chat Assistant:** Guides customers throughout their shopping journey by answering questions, recommending products, explaining website features, and providing personalized assistance.
    
- **Frequently Bought Together Bundles:** Administrators can create product bundles based on common purchasing patterns. These bundles are displayed throughout the website and can also be proactively recommended by the AI assistant during conversations.
    
- **Customer Retention Pipeline:** Collect customer emails and build automated communication workflows for promotions, abandoned carts, order follow-ups, and personalized marketing campaigns.
    
- **Flexible Payment Options:** Support multiple payment methods to accommodate different customer preferences, including:
    
    - Cash on Delivery (COD)
        
    - CIB (interbank) payment
        
    - Visa/Mastercard payments
        
    - Reservation with payment upon in-store pickup
        
    - Installment payment facilitation by allowing customers to upload the required supporting documents
